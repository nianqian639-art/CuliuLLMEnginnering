# Prompt Eval Report Draft

## 1. 任务定义

本项目的任务是：根据游戏当前 `snapshot`，为玩家生成下一步候选落子建议，并通过二次评估提升建议的可信度。

## 2. Prompt 版本说明

- `zero_shot`：只给任务说明
- `one_shot`：补 1 个示例
- `few_shot`：补 2 个示例，并更强调 reason/risk 的表达

本轮对比实验唯一主变量是 `prompt_version`。

## 3. 测试样本

样本来自两类：

- 实时房间样本：验证真实接口链路
- 静态快照样本：补齐边界场景与失败案例

覆盖场景：

- 中局
- 稀疏盘面
- 密集盘面
- 含 `X`
- 非标准尺寸
- 登录失败 / 房间身份失败
